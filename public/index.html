Flinders Timetable Parser

Now updated for Sem 1 2026!

A very simple, unofficial tool for automatically exporting your Flinders University timetable into iCal files that you can import into Google Calendar, Outlook, Apple Calendar, etc.
Read the steps below for how to use, and please read the disclaimers for some important info.
Created because, for some reason, during the massive revamp of the student system, the uni (in their infinite wisdom) decided not to add calendar exports.
Please Flinders, add this officially. You could even make it sync! I wouldn't use any of the code you find here though, bad idea.

Steps

First go to Compass and make sure your session is saved in your browser cookies.

Then go to this link and download its contents. It should download as a .json file.

Upload that file below, select what extras you want, and hit generate. It'll generate you an iCal file (or files) that you can import into your calendar software of choice.

Sometimes you'll get an error (Likely a 401) when trying to fetch your timetable. A known fix is clearing the cookies on Compass and trying again. If you can view your timetable in Compass then the timetable API link should work too.

Disclaimers

I take no responsibility for any incorrect dates published! I make these edits in stolen moments and very rarely proof them. This thing is so unofficial it would shock you.

That being said, all info (except for your classes) is taken from the yearly FUSA calendars, so thank them next time you see them!

This program runs fully client-side! Meaning: nothing "calls home", no data gets sent out, I can't see anything you do with your timetable. It all runs in your browser!

You can see for yourself by opening Developer Tools, going to the network section, and monitoring incoming and outgoing traffic. Other than loading this page itself, no other calls should be made.

If you're still unconvinced, download it and run it offline :D

Additionally, I have no plans to maintain this forever, but the code is all on this one page (and on my GitHub repo) so go wild if you feel like it.
